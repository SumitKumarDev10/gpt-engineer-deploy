Getting Started

To get started with GPT-Engineer, follow the instructions below:

  1. Clone the GPT-Engineer repository: git clone https://github.com/AntonOsika/gpt-engineer.git
  2. Install the required dependencies: pip install -r requirements.txt
  3. Explore the codebase and documentation to learn how to use GPT-Engineer in your projects.
