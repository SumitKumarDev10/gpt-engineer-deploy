Getting Started with GPT-Engineer

To get started with GPT-Engineer and unleash the power of AI-assisted code generation, follow the steps below:

  1. Choose either the stable or development release.
  2. If you opt for the stable release, run the following command:
    3. pip install gpt-engineer
  3. If you prefer the development version, clone the repository and install it:
    4. git clone https://github.com/AntonOsika/gpt-engineer.git cd gpt-engineer pip install -e .
  4. Set up your OpenAI API key (preferably with GPT-4 access). For example:
    5. export OPENAI_API_KEY=[your api key]

    (Alternative for Windows: set OPENAI_API_KEY=[your api key] on cmd, or $env:OPENAI_API_KEY="[your api key]" on PowerShell)

  5. Create a new project folder by copying the example project:
    6. cp -r projects/example/ projects/my-new-project
  6. Edit the prompt file in your new project folder to specify your code generation requirements.
  7. Run GPT-Engineer on your project:
    8. gpt-engineer projects/my-new-project
